Quick help:

Watch: enables watching that message class. If unchecked, the class won't be tracked altogether and won't count towards total count.

Track new: counts new messages since you last opened your message center, and optionally alerts you about them.

Badge alert: makes badge on the extension icon change to green when there are new messages of that type.

Sound alert: plays the notification sound when new messages of this type are detected. Global setting overrides this.

Desktop notification: shows a desktop notification popup when new messages of this type are detected. Global setting overrides this.
