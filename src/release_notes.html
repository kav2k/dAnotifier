An apology

To everyone, I'm sorry that fixing the latest serious bugs took almost 3 weeks.

I am usually much swifter with bug-squishing. Believe me, the actual bug took less than 5 minutes to fix.

The problem was that I could no longer simply patch that bug. Due to some changes to the Chrome Web Store, the extension needed deep reworking to be accepted. All this work was already done in the Beta version, but I needed to shape it up for general release.

Then real life knocks at your door and you are suddenly much more busy that you'd like to be.

But now the wait is over, Beta is wrapped up as v7, and you get both the much-needed fix and new features, such as the new interface. Read on below to get to know them.

There are other things I hope to get done very, very soon. For example, you don't need to ask about making the notification go away automatically, I'll be working on it shortly. But getting this fix out was urgent, so here you have it. I apologize again for the delay.

dA Message Notifier release notes

Proceed to the Options page to configure new features, and check out the manual to get to know them.

0.7.0.0

[FIXED] my.deviantart.com subdomain errors: messages now properly register as read, and tabs are reused.

[ADDED] "Brief" tooltip mode. Fixes long-standing problem with cut-off tooltips on Windows.
