dA Message Notifier release notes

Proceed to the Options page to configure new features, and check out the manual to get to know them.

0.7.0.4

[FIXED] Group watch preferences didn't really work before this.

0.7.0.3

[CHANGED] Desktop Notification update mechanism simplified. [ADDED] The extension should now update immediately if it has an update waiting.

0.7.0.2

[FIXED/REMOVED] Removed "News articles" message class that deviantART retired (fixing a critical bug)

0.7.0.1

[ADDED] An option (opt-in) to ignore Promoted content.

0.7.0.0

[FIXED] my.deviantart.com subdomain errors: messages now properly register as read, and tabs are reused.

[ADDED] New popup interface. Enabled by default, you can return to old behavior in options.

[ADDED] "Brief" tooltip mode. Fixes long-standing problem with cut-off tooltips on Windows.

[ADDED] Sanity check: notifications are disabled if not supported (Chrome 28+).

[FIXED] CSS bugs across all pages.

..and lots of internal changes.
